LaoLin的网站系统源代码【a9】
======

## 所需子模块:
* ~LazyPHP (基于EazyChen的LazyPHP修改来的网站的PHP框架)
* a9-linjianping (服务器端主要代码，基于lazyPHP)
* a9-static (客户端静态文件：css,js,img)

## Quick start / 安装方法
* 方式1 源代码放在网站目录下
* 方式2 源代码放在别的目录，在网站目录下做两个符号链接：
    + index.php符号链接到代码主目录的index.php
    + static符号链接到代码主目录的static目录
